# TripPing: 사용자 요구 기반 당일치기 최적화 확장형 관광 코스 추천 서비스

> 2026 관광데이터 활용 공모전

## 서비스 소개

기존 관광 추천 서비스는 동행자 구성, 여행 목적 등 사용자의 다양한 여행 상황을 충분히 반영하지 못하는 한계가 있습니다. 예를 들어 대학생과 초등학생 동생이 함께 나들이를 계획할 때 '아이 동반' 조건으로 검색하면 키즈카페나 박물관 같은 획일적인 장소만 추천되는 식입니다.

본 서비스는 자연어 입력과 선택형 조건을 결합해 사용자 요구를 정밀하게 분석하고, 핵심 관광지를 중심으로 주변 코스를 단계적으로 확장하는 **거점 기반 확장형 추천 구조**를 통해 당일치기 여행에 최적화된 맞춤형 코스를 제공합니다.

## 주요 기능

| 기능 | 설명 |
|---|---|
| 조건/자연어 기반 관광지 추천 | 여행 타입(자연/도시/복합), 연령, 동행자(연인/아이/가족), 지역 등 선택 조건 + 자유 텍스트 입력 |
| 주요 관광지 선택 및 확장 추천 | 주요 관광지 3곳 추천 → 사용자가 선택한 곳을 중심으로 주변 명소·식당·카페 확장 추천 |
| 코스 카드 생성 및 공유 | 확정 경로를 카드 형태로 시각화, 저장 및 SNS 공유 지원 |
| 코스 카드 보관함 | 생성된 코스 카드를 한눈에 모아보는 보관함 제공 |

**서비스 흐름**: 여행 조건 입력 → 추천 관광지 선택 → 코스 카드 생성


## 차별성

- 필터 기반 나열식 추천이 아닌, 자연어+조건 결합으로 사용자 의도를 정교하게 반영
- 추천 결과를 단순 수용하는 대신 사용자가 직접 관광지를 선택하고 코스를 확장해 나가는 능동적 참여 구조
- 효율적 동선의 당일치기 코스 + 직관적인 카드형 시각화 및 공유


## 발전 방향

- 대화형 인터페이스 도입으로 실시간 요구사항 반영
- 코스 카드 평점·피드백 기능 추가
- 보관함 카드형/지도형 UI 선택 제공
- 코스 공유·평가 커뮤니티 기능 확장
- 축적 데이터 기반 개인화 추천 알고리즘 고도화

## 기대효과

- 제한된 시간 내 효율적인 당일치기 여행 계획 지원, 사용자 만족도 향상
- 거점 확장형 구조를 통한 맥락 기반 여행 코스 제공
- 카드형 공유 콘텐츠 확산으로 숨겨진 관광지 발굴 및 지역 관광 활성화
